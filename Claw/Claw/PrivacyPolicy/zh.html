【引言】

欢迎您使用小龙虾部署助手（以下简称“本应用”）。我们深知个人信息对您的重要性，始终尊重并致力于保护您的隐私安全。本政策旨在说明我们如何处理您的相关信息。

请注意：本应用主要提供小龙虾（OpenClaw）的配置教程与部署指引，不提供自动化脚本的本地运行环境，亦不收集您的任何操作行为数据。

一、我们如何收集和使用您的信息

我们将遵循正当、合法、必要的原则，仅为实现本政策所述的指引服务目的收集信息：

  1. 保障服务安全运行：为识别异常状态及保障应用安全，我们会收集您的设备信息（如设备型号、操作系统版本、唯一设备标识符、登录IP地址等）。这些信息本身无法识别特定自然人身份。
  2. 配置教程指引：您在使用本应用查看配置教程时，我们不会记录您的相关点击操作记录。应用内的指引内容旨在辅助您在外部环境（如您的个人服务器或终端）进行部署。
  3. 数据处理说明：应用内涉及的技术参数仅供参考，我们不会收集您在外部部署过程中产生的任何私钥、Token或配置参数。

二、Cookies 及同类技术的使用

为确保应用正常高效运转，我们可能会在您的设备上存储小型数据文件。您可以根据自身需求通过设备设置拒绝或清除相关数据，但这可能会影响部分内容的加载效率。

三、我们如何共享、转让或披露您的信息

  1. 共享：除法律规定或获得您的明确同意外，我们不会与任何第三方共享您的个人信息。
  2. 第三方SDK：为保障应用稳定性，我们可能接入统计分析类SDK。其处理行为受其自身隐私政策约束，我们建议您在使用前查看相关协议。
  3. 公开披露：我们仅在法律强制要求或获得您明确授权的情况下，才会披露您的相关信息。

四、我们如何保存和保护您的信息

  1. 存储地点：收集的信息存储于中国境内。
  2. 保护措施：我们采用符合业界标准的安全防护措施，防止您的信息被未经授权的访问或泄露。由于互联网并非绝对安全，请妥善保护您的设备。

五、未成年人保护

本服务主要面向成年开发者。如您为未成年人，请在监护人同意后再阅读本教程。我们将依据法律规定对未成年人信息进行特殊保护。

附件：小龙虾部署助手权限申请与使用说明

为保障功能的实现与安全运行，我们会申请以下操作系统权限。您可以根据需要随时在系统中关闭相关权限：

  • 1. Privacy - Push Notifications Usage Description (消息通知推送)
    目的：用于向您推送配置教程更新提醒或重要安全公告。
    拒绝后果：无法使用网络
  • 2. Privacy - Network Extensions Usage Description (网络权限)
    目的：用于从服务器获取最新的部署教程、常见问题解答及访问隐私政策页面。
    拒绝后果：无法正常查看教程内容。
  • 3. Privacy - Tracking Usage Description (跟踪权限)
    目的：为了提升您在使用产品过程中的体验，降低内容广告对您的打扰与影响，我们需要您允许APP获取您的设备标识符（IDFA）。
    拒绝后果：我们无法分析程序发生错误原因。
